## 🔧 Installation and Setup

holy-c-telegram-bot does not require installation beyond running the executable. Here is how to set up your system for first use:

### Set your Telegram bot token

To make this bot work, you must have a Telegram bot token. This token connects your bot to Telegram servers.

- Open Telegram and search for the “BotFather” user.
- Type `/newbot` and follow instructions to create a new bot.
- After setup, BotFather will give you a token. Copy this token.

### Add token to the bot

1. Create a plain text file named `config.txt` in the same folder where you run the bot.
2. Paste your bot token into this file and save it.
3. Run the bot executable file again.
4. The bot will read the token and connect to Telegram.

If the token is valid, the bot will be fully functional.
